SUMMARY
-------

Remote Resource Solution Pack for Islandora

Loads all required Fedora Objects, and creates an empty collection
object to accept links to remote resources.

REQUIREMENTS
------------

Apache Tika
  To determine mime types and extract meta-data.
  Also used to optionally extract text to store for indexing to provide text searching within the document.
  http://tika.apache.org/
